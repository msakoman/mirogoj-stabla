Postavljanje aplikacije – jednokratno

Ova aplikacija koristi Firebase za bazu podataka i Google Maps za kartu. Obje usluge imaju besplatni tier koji je dovoljan za ovaj projekt.

🔥 Korak 1 – Firebase
Idi na console.firebase.google.com i klikni Add project. Imenuj projekt (npr. mirogoj-stabla).
U izborniku lijevo → Build → Authentication → Get started → Email/Password → Enable → Save.
Authentication → Users → Add user: upiši email i lozinku za prijavu u aplikaciju.
Build → Firestore Database → Create database → Start in production mode → odaberi europe-west3 (Frankfurt).
Firestore → Rules → zamijeni sve s kodom ispod → Publish: 
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /trees/{treeId} {
      allow read: if true;
      allow write: if request.auth != null;
    }
  }
}
Build → Storage → Get started → Start in production mode → odaberi europe-west1. Zatim Storage → Rules → zamijeni s: 
rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /trees/{imageId} {
      allow read: if true;
      allow write: if request.auth != null
                   && request.resource.size < 2 * 1024 * 1024
                   && request.resource.contentType.matches('image/.*');
      allow delete: if request.auth != null;
    }
  }
}
Project Settings (zupčanik gore lijevo) → Your apps → </> (Web) → Registriraj app → kopiraj firebaseConfig objekt.
🗺️ Korak 2 – Google Maps API ključ
Idi na console.cloud.google.com → odaberi isti projekt ili kreiraj novi.
APIs & Services → Library → traži "Maps JavaScript API" → Enable.
APIs & Services → Credentials → Create Credentials → API key. Kopiraj ključ.
⚙️ Korak 3 – Unesi ključeve u aplikaciju

Otvori index.html u tekst editoru, pronađi sekciju // ===== KONFIGURACIJA ===== i zamijeni placeholder vrijednosti s pravim ključevima.
